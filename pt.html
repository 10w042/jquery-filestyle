Descrição

O filestyle-bootstrap.js é um plugin jquery para personalizar input file usando o Bootstrap.



Requisitos

Bootstrap.css

jquery.js



Instalação

Inclua o arquivo bootstrap-filestyle.js no seu código.



Usando

Básico:

$(":file").filestyle()


Usando opções:

$(":file").filestyle({
    textField: Arquivos,
    icon: true
})



Options

buttonText String Personalização do texto do botão.
textField Bool Mostra ou esconde o campo de texto.
classButton String Use classes Bootstrap no botão.
classText String Use classes Bootstrap para o campo de texto.
icon Bool Adiciona um icone no botão.
classIcon String Use of classes Bootstrap para o icone.


Methods

clear String Limpa um filestyle definido.
