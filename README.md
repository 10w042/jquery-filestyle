# Filestyle Bootstrap

Customization of input html file for Bootstrap Twitter

## Getting Started

In your web page:

```html
<script src="jquery.js"></script>
<script src="Filestyle-bootstrap.min.js"></script>
<script>
jQuery(function($) {
  $(':file').filestyle();
});
</script>
```
